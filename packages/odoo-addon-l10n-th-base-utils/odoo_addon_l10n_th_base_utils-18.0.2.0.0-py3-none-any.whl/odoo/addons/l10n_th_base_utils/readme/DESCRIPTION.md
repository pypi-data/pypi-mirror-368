This module is a utility package for Thailand, providing helper functions commonly used in Thai-specific contexts.

It includes features such as:

- Conversion between Arabic and Thai numerals
- Thai month name and Buddhist calendar (Thai year) conversion
- Thai font support:

    - Font `THSrisakdi`
    - Font `THSarabunNew`
    - Font `THSarabun`
    - Font `THNiramitAS`
    - Font `THMaliGrade6`
    - Font `THKrub`
    - Font `THKoHo`
    - Font `THKodchasal`
    - Font `THK2DJuly8`
    - Font `THFahkwang`
    - Font `THCharmonman`
    - Font `THCharmofAU`
    - Font `THChakraPetch`
    - Font `THBaijam`
    - Font `AngsanaNew`
    - Font `Webdings`
