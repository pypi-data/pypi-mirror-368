Installation
============


Install from PyPI
-----------------

.. note::

    CASM is built for macOS x86_64 (Intel), macOS arm64 (Apple Silicon), Linux x86_64, and Linux aarch64.

The latest release of casm-project can be installed with:

    pip install casm-project
